## üß≠ Project Overview  

Recruiters spend hours scanning resumes for relevant skills. This project solves that by building an **Automated Resume Screening Tool** that:  

- Parses resumes (PDF, DOCX, TXT)  
- Extracts skills, experience, and education  
- Ranks candidates using a **hybrid scoring model** (rules + embeddings)  
- Provides explainable results and a recruiter-friendly web dashboard  

This project is designed to be **industry-oriented**, **fully runnable**, and **ready for deployment** on GitHub and cloud platforms.
